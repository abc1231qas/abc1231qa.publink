## 🎨 設計改進內容

### 1. **背景與質感**
- ✅ 從現代化的藍白漸層改為米白色紙張質感背景
- ✅ 加入微妙的徑向漸層效果，營造紙張紋理

### 2. **字體系統**
- ✅ 從 Sans-Serif 系統字體改為 **Noto Serif TC**
- ✅ 統一字重：300（細體）、400（正常）、500（中等）
- ✅ 增加字距（letter-spacing）以提升閱讀體驗

### 3. **配色方案**
**之前（現代風格）**:
- 主色：亮藍色 `#0070f3`
- 警告色：亮紅色 `#ff4d4f`
- 背景：白色 `#ffffff`

**現在（Zen 美學）**:
- 墨色：`#2C2C2C` (ink-black)
- 深灰：`#333333` (text-deep)
- 中灰：`#595959` (text-mid)
- 金色：`#C5A065` (gold-muted)
- 淡金：`rgba(197, 160, 101, 0.15)` (gold-light)

### 4. **UI 元件重設計**

#### 輸入框
- 移除圓角 `border-radius`
- 使用細邊框 `1px solid var(--border-subtle)`
- 半透明背景 `rgba(255, 255, 255, 0.8)`
- Focus 時邊框變為金色

#### 按鈕
- 從實心按鈕改為邊框按鈕
- 移除所有顏色填充
- Hover 時顯示金色淡背景（滑動效果）
- 統一使用 `::before` 偽元素實現動畫

#### 列表項目
- 移除卡片陰影
- 使用細分隔線
- Hover 時顯示淡金色背景
- 箭頭符號改為金色

### 5. **動畫效果**
- ✅ 加入整體淡入動畫（fadeIn）
- ✅ 按鈕 Hover 滑動效果
- ✅ 平滑的顏色過渡

## 🔍 技術細節

### CSS 變數系統
```css
:root {
  --bg-rice: #F7F7F5;
  --ink-black: #2C2C2C;
  --text-deep: #333333;
  --text-mid: #595959;
  --gold-muted: #C5A065;
  --gold-light: rgba(197, 160, 101, 0.15);
  --border-subtle: rgba(44, 44, 44, 0.15);
}
```

### 關鍵動畫
```css
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### 按鈕 Hover 效果
```css
button::before {
  content: '';
  position: absolute;
  left: -100%;
  background: var(--gold-light);
  transition: left 0.5s ease;
}

button:hover::before {
  left: 0;
}
```
