# 網站風格改造對比

## 🔄 改造概覽

本次改造將網站從**現代科技風**轉變為**東方禪意極簡風**，以下是詳細的前後對比。

---

## 🎨 配色對比

### 改造前 (現代科技風)
```css
--primary: #2d3436    /* 深灰 */
--accent: #0984e3     /* 亮藍色 */
--bg: #dfe6e9         /* 淺灰藍 */
```
**特點**：科技感、冷色調、高對比

### 改造後 (禪意風)
```css
--bg-rice: #F7F7F5    /* 米白 */
--ink-black: #2C2C2C  /* 墨黑 */
--gold-muted: #C5A065 /* 霧金 */
```
**特點**：溫暖、自然、低飽和度

## 🎬 動畫效果對比

### 改造前
```css
/* 簡單的 hover 效果 */
.card:hover {
  transform: translateY(-5px);
  transition: 0.3s;
}

.btn:hover {
  background: #0051a2;
  transition: 0.2s;
}
```

### 改造後
```css
/* 漸進式淡入動畫 */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 元素依序出現 */
.enso { animation: fadeIn 1.2s ease-out 0.3s forwards; }
h1 { animation: fadeIn 1s ease-out 0.6s forwards; }
.subtitle { animation: fadeIn 1s ease-out 0.9s forwards; }
/* ... 以此類推 */

/* 細膩的 hover 效果 */
.zen-link::before {
  background: var(--gold-light);
  transition: left 0.5s ease;
}
```

## 📱 響應式設計對比

```css
  max-width: 400px;
}
```

```css
/* 更細緻的斷點調整 */

  h1 { font-size: 2rem; }

}
```
